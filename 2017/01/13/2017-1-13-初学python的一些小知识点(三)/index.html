函数的参数

​ Python的函数定义非常简单，但灵活度却非常大。除了正常定义的必选参数外，还可以使用默认参数、可变参数和关键字参数，使得函数定义出来的接口，不但能处理复杂的参数，还可以简化调用者的代码。

  • 位置参数

    x和y，这两个参数都是位置参数，调用函数时，传入的两个值按照位置顺序依次赋给参数x和y。

  • 默认参数

    当一个函数的参数可以有默认值的时候就需要一个默认参数

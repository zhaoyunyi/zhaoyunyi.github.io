Hexo博客-如何文章里面放入图片

发表于 2017-02-04 | 更新于 2021-03-28

​ 在用markdown写文章的时候如果想在文章里面插入图片的话应该怎么做呢？

  • Hexo全局图片

    Hexo项目目录下有一个明显的图片资源目录source/images，这里适合放一些项目全局都要用到的图片，当然放文章的图片也可以，然后通过markdown语法使用它。

  • 每篇文章图片

    修改配置_config.yml中的属性post_asset_folder之后，执行命令$ hexo new post_name，在source/_posts中会生成文章post_name.md和同名文件夹post_name。将图片资源放在post_name中，文章就可以使用相对路径引用图片资源了。

  • CDN图床

    找一个免费的图床网站服务，上传完需要展示的图片之后，会生成对应URL链接，最后将对应的链接并粘贴到文章里即可。

Mac Python 安装mysqlclient遇到的问题

Pipenv install mysqlclient

  • 出现错误之后按照官网的提示进行修改，上面有Mac安装需要做的操作。除了Mac环境以外还有其他环境的可以查阅。

  • 按照提示 brew install mysql-connector-c 之后会有 brew unlink的提示，按照提示进行操作，然后完成安装。

  • 之后再进行，Pipenv install mysqlclient 还会有错误，这时因为没有openssl环境变量，这样做

    export LDFLAGS=”-L/usr/local/opt/openssl/lib”

    export CPPFLAGS=”-I/usr/local/opt/openssl/include”

碰到的其他问题

　　运行项目抛出以下错误_mysql_exceptions.OperationalError: (2059, ) 因为8.0之后的数据库新增了用户用户密码，老的驱动还没跟上，所以需要修改用户密码格式信息,。

1.登录mysql，连接用户为root。

mysql -u root -p

2.执行命令查看加密方式

use mysql;
select user,plugin from user where user=’root’;

3.执行命令修改加密方式

alter user ‘root‘@’localhost’ identified with mysql_native_password by ‘yourpassword’

4.属性权限使配置生效

flush privileges

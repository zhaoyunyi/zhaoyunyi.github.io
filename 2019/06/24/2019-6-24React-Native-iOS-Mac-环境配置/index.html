React Native iOS Mac 环境配置

当前大致的开发环境

  • Mac 电脑 10.14.5
  • Xcode Version 10.2 (10E125)
  • 科学上网的环境
  • 安装了常用的brew、python、node12.0.1 版本等

安装React Nactive 开发环境步骤

　　基本的安装步骤和流程参考以下链接搭建开发环境，从中主要查看iOS相关的即可。里面有个关键是记得在Xcode——Perference——Locations——Command Line Tools 选项选上对应的Xcode版本。其他的通过命令行安装，注意有些命令需要科学上网环境。

期间碰到的问题

  1. 完全新建的工程只要保证科学上网环境，或者项目的包的依赖下载路径是国内的有效镜像就没有问题。
  2. 中间在 npm install 或者yarn install 安装的过程中出现一部分安装成功一部分安装失败的问题，我在安装了webStorm之后，在WebStrom 的命令行里莫名其妙的安装成功了，可能只是一时的网络问题。
  3. 一直卡在 “install third party”，是因为React Native 需要安装几个依赖的库，下载非常慢（哪怕是有科学上网，我也失败了）。React Native进行依赖库下载的执行脚本，选择对应自己安装的对应的React Native 版本，然后在脚本最下面有几行下载资源的操作，自行进行下载。
    下载完之后，将压缩包文件放入 ~/.rccache 目录下 mkdir ~/.rncache，然后在命令行里进行运行react-native run-ios。如果有一些奇怪的问题就再次 npm install 之后进行运行。具体操作可以，参考1，参考2。
  4. 如果项目还依赖一些公司私库或者公司私库做了镜像的，可以在项目目录下对 npm或者yarn的 请求下载链接进行配置。
